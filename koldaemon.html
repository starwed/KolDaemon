Koldaemon firefox extension

Koldaemon is a tabbed chat implementation for the Kingdom of Loathing.

Install Me!

Requires firefox, but does not require a restart. Access preferences through the "Settings" button.

Features...

  • Get in chat or growl style notifications for PMs.
  • Cute little headers on PM conversations.
  • Customize what words draw your attention to a message.
  • Split Huggler radio and Biggus into channel pvp/radio

Antifeatures...

  • Probably has missing functionality compared to ActiveChat, but let me know what's missing!
  • Almost definitely not compatible with other chat scripts, but see above!
  • Occasionally chat won't load correctly -- refresh KoL to fix this. (And message me if you figure out how to reliably reproduce it!)
